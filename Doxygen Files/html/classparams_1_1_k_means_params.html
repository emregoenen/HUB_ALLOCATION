Detailed Description

Dataclass for holding parameters of clustering algorithm Initialized by default values.

Parameters
n_clustersThe number of clusters to form as well as the number of centroids to generate.
initNumber of time the k-means algorithm will be run with different centroid seeds. The final results will be the best output of n_init consecutive runs in terms of inertia.
max_iterMaximum number of iterations of the k-means algorithm for a single run.
algorithmK-means algorithm to use.
